# CAS-782-Digital-Twin-final-repo

This repository contains the code, data, outputs, and report for the CAS 782 R&D project:
"A Lightweight Data Quality Module for Digital Twin Data Pipelines."

## Files
- `01_air_quality_pipeline.ipynb`: main notebook
- `AirQualityUCI.csv`: raw dataset
- `standardized_air_quality.csv`: standardized dataset
- `cleaned_air_quality.csv`: cleaned dataset
- `validation_report.csv`: validation summary
- `repair_log.csv`: repair summary
- `quality_comparison.csv`: raw vs cleaned comparison
- `*.png`: figures used in the report

## How to run
Open the notebook and run all cells in order.

## Environment
Python with pandas, numpy, and matplotlib.
